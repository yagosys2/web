how to put ssh into supervisor



ubuntu@VM-0-7-ubuntu:/etc/supervisor/conf.d$ cat ssh.conf

[program:ssh]

command=/usr/bin/ssh -D:6005 root@23.105.223.133 -p 27262

autostart=true

log_stdout=true

log_stderr=true

user=root

startsecs=60

thing you have to prepare:

  • generate ssh_key on the client with ssh-keygen
  • use ssh-copy-id to the remote host
  • check error code on supervisor log ,one of the possible reason of 255 means remote site may not have root privildege to create a tunnel
